<div class="user-html user-html__with-lazy-load"><h3 id="item-description__luzen-architecture-and-interior-design-html-template"><strong>Luzen — Architecture &amp; Interior Design HTML Template</strong></h3>
<p><strong>Luzen</strong> is a premium, fully responsive HTML5 &amp; CSS3 template built for architecture studios, interior designers and creative practices. It ships with a <strong>7-page layout</strong> — Home, About, Services, Projects, Project Detail, Contact and Team Member — sharing one quiet, editorial design system and powered by GSAP + ScrollTrigger motion, a hand-built WebGL hover effect and Lenis smooth scrolling.
<br><br>Looks great on all screen sizes and devices. Fully responsive, fast loading, and easy to customize without deep coding knowledge.</p>

<h3 id="item-description__key-features">Key Features</h3>
<ul>
<li>7-page architecture &amp; interior design template on one shared design system</li>
<li>GSAP + ScrollTrigger motion: cross-fading hero slider, scroll reveals, animated counters</li>
<li>Custom WebGL displacement hover, hand-built with Three.js</li>
<li>Web3Forms-powered contact form — no backend required</li>
<li>Fully responsive design and fast loading across devices</li>
</ul>
<h3 id="item-description__why-choose-luzen">Why choose Luzen?</h3>
<ul>
<li>
<strong>7-Page Architecture &amp; Interior Design Template</strong>
<p>Home, About, Services, Projects, Project Detail, Contact and Team Member — seven pages sharing one cohesive editorial design system.</p>
</li>

</li>
<li>
<strong>Custom WebGL Displacement Hover</strong>
<p>A hand-built Three.js shader hover effect for an extra layer of polish on imagery, with no third-party hover plugin.</p>
</li>
<li>
<strong>Lenis Smooth Scrolling</strong>
<p>Buttery, inertia-based page scrolling throughout the template.</p>
</li>
<li>
<strong>Web3Forms Contact Form</strong>
<p>The contact form submits directly via Web3Forms — no PHP or mail server needed. Includes honeypot spam protection and clear success/error messaging.</p>
</li>
<li>
<strong>Magnetic Buttons &amp; Custom Cursor</strong>
<p>Interactive magnetic button hovers and a custom cursor reinforce the studio's editorial feel across every page.</p>
</li>
<li>
<strong>Easy Customization</strong>
<p>SCSS source included with a clean 7-1 architecture and design tokens for colors, fonts and spacing.</p>
</li>
<li>
<strong>Completely Responsive &amp; Accessible</strong>
<p>Tested across five breakpoints (1200 / 1024 / 992 / 768 / 480px) with <code>prefers-reduced-motion</code> support built in.</p>
</li>
<li>
<strong>Fast Loading Speed</strong>
<p>No jQuery, no build step required to run, locally vendored libraries and no unnecessary dependencies.</p>
</li>
</ul>
